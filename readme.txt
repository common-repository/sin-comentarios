Deshabilita los comentarios de Wordpress por completo.
 
== Description == 
 
Deshabilita los comentarios de Wordpress por completo. Inhabilita los formularios de comentarios en el front-end, oculta los comentarios existentes, elimina los trackbacks, los menús y enlaces en el panel de administración.
 
== Installation == 
 
1. Descargar el archivo "sin-comentarios.zip" y subir a Wordpress con el administrador de plugins.
2. Activar el plugin desde el administrador de plugins de wordpress para deshabilitar los comentarios.
 
== Frequently Asked Questions == 
 
= ¿Es un plugin gratuito? = 
 
Totalmente y lo seguirá siendo. 
 
= Cómo funciona = 
 
Simplemente activa el plugin. No necesita ajustes.

= ¿Elimina los comentarios? =

No elimina los comentarios existentes, sólo desahabilita la función. Desactivando el plugin, se volverá a activar tanto el menú con los comentarios guardados como los formularios de comentarios en el frontal.
